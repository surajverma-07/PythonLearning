# Object Types / Data Types

- Number : 1234, 3.14, 3+5j, 0b132, Decimal(), Fraction()

- String : 'suraj', "Bob's", b'a\x01c', u'sp\xc4m'

- List : [1, [2, 'three'], 4.5], list(range(10))

- Tuple : (1, 'spam', 4, 'U'), tuple('suraj'), namedtuple

- Dictionary : {'fruite' : 'banana', 'taste':'yum'}, dict(hours=10)

- Set : set('abc'), {'a','b','c'} // Unique elements

- File : open('exm.txt'), open(r'C:\ham.bin', 'wb')

- Boolean : True, False

- None : None 

- Functions, modules, classes 

- Advance : Decorators, Generators, Iterators
